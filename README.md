# Elegance Salon - Full Stack Management System

A dynamic, database-driven web application built to streamline salon operations, customer appointments, and administrative management. This project was built using **Core PHP** and **MySQL** to practice solid backend architecture, session management, and relational database design.

## Key Features
* **User Authentication:** Secure login and registration system for customers and administrators using PHP sessions.
* **Appointment Booking System:** Dynamic booking form where customers can select services, dates, and available time slots.
* **Admin Dashboard:** A comprehensive backend panel allowing admin users to:
  * View, approve, or cancel incoming appointments.
  * Manage salon services (Add, Edit, Delete prices and descriptions).
  * View registered user data.
* **Responsive UI:** Fully responsive design built with **Bootstrap**, ensuring a seamless experience across mobile, tablet, and desktop devices.

## Tech Stack Used
* **Backend:** Core PHP
* **Database:** MySQL (Relational Database Design)
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
* **Tools:** Git, GitHub

## How to Setup Globally/Locally
1. Clone this repository: `git clone https://github.com/YOUR_USERNAME/elegance-salon.git`
2. Import the `saloon.sql` file into your local phpMyAdmin.
3. Rename `config-copy.php` to `config.php` and add your local database credentials.
4. Run via XAMPP/Apache local server.
